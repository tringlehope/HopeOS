# HopeOS
это конфигурация хайперленд, которую может использовать или модернезировать каждый. много всего брал у "Zproger"

тут все просто: копируешь мой конфиг и подставляешь в свои папки. (рекомендую старые файлы переименовать, а мои конфиги вставить в новые текстовые документы)

я использую программы:
1. rofi-wayland
2. waybar
3. kitty
4. swaybg
5. nemo
6. mako
(для арч так ставятся нужные шрифты: "sudo pacman -S ttf-jetbrains-mono-nerd ttf-font-awesome")

особенности:
1. я использую два монитора: DP-5 и HDMI-A-2.  если у вас другие мониторы с другим разрешением поменяйте на свое. (hyprctl monitors - команда для того что бы узнать про свои мониторы)
